# Hero Media Folder

Place background videos and images for the hero section here.

## Usage

In your Google Sheet (`hero` tab), set `media_url` to:

```
/media/hero/your-video-filename.mp4
```

or for a hosted Cloudinary URL, paste the full URL directly.

## Recommended formats

- Video: MP4 (H.264), keep under 20MB for fast loading. For larger hero videos, prefer Cloudinary or another hosted media service instead of committing the file to git.
- Image: JPG or WebP, 1920x1080 or wider

## Example filenames

- `memorial-day-kickoff.mp4`
- `juneteenth-night.mp4`
- `memorial-day-kickoff-hero.jpg`
