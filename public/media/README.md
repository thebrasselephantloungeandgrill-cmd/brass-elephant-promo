# Event Media Directory

Place event images and videos in this directory.

## Required Files

### Memorial Day Weekend Kickoff
- `memorial-day-flyer.jpg` - Event flyer image (used for hero and gallery)

### Juneteenth Night
- `juneteenth-hero.jpg` - Hero background image
- `juneteenth-flyer.jpg` - Event flyer image

## File Naming Convention

Use the event slug as the prefix for all media files:
- `{event-slug}-hero.jpg` - Hero background image
- `{event-slug}-flyer.jpg` - Event flyer
- `{event-slug}-hero.mp4` - Hero background video (optional)
- `{event-slug}-recap.mp4` - Recap video (optional)

## Image Guidelines

- **Hero images**: Recommended size 1920x1080px or larger
- **Flyer images**: Recommended size 1080x1920px (vertical)
- **Gallery images**: Recommended size 1200x800px or larger
- **Format**: JPG or PNG for images, MP4 for videos
- **File size**: Keep under 5MB for optimal loading

## Placeholder Behavior

If an image file is missing, the component will display a styled placeholder with the event title or media description.
